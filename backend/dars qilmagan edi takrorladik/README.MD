# Azizbek aka dars qilmagandi darslarni takrorladik

## fayl yaratish uchun quyidagi buyruqni terminalga yozing:
```
npm init -y
```
## bu buyruq package.json faylini yaratadi

## expressni o'rnatish uchun quyidagi buyruqni terminalga yozing:
```
npm install express
```

## expressni ornatganimizdan keyin uni server.js faylida import qilishimiz kerak va u bizga node_modules papkasi ichidan keladi.

## gitignore faylini yaratishimizning asosiy sababi biz uni ichiga havfsizlik uchun ochilgan filelarni qoshib yubomaslik va node_modules papkasini qoshib yubormaslik uchun kerak.
